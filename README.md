  - Other...

## Game Control:
- **A / LeftArrow**: For moving character to the left.
- **D / RightArrow**: For Moving character to the right.
- **W / Space / UpArrow**: For using the zet of character.
- **K**: Debug Mode (Currently Shows Red overlay over the entities)
- **F**: Shoot Bullet.
